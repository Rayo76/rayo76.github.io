# Universal LLM Prompt Builder V4

A powerful, beautiful, and **fully client-side** prompt engineering tool that helps you create highly structured, optimized prompts for ChatGPT, Claude, Gemini, and other LLMs.

Built for efficiency — **fewer tokens, better outputs, lower costs**.

![Universal LLM Prompt Builder](https://via.placeholder.com/800x400/0d7a72/ffffff?text=Universal+LLM+Prompt+Builder+V4)

## ✨ Features

- **Three Prompt Types**: Text, Image, and Video prompting
- **Smart Category Templates**: Auto-fills optimal placeholders based on 10+ categories (Travel, Health, App/PRD, Resume, Debugging, etc.)
- **Multi-LLM Ready**: One-click copy prompts optimized for:
  - ChatGPT
  - Claude (Text only)
  - Gemini
  - Generic (works with any LLM)
- **Token Estimation**: Real-time prompt token counting
- **Anti-Hallucination & Step-Locking Guards**
- **Clean JSON Export** for developers and automation
- **Fully Responsive** modern UI
- **Privacy-First**: Everything runs in your browser — **no data is sent to any server**

## 🛠️ How to Use

1. Open `index.html` in your browser (or deploy to GitHub Pages/Netlify)
2. Choose your prompt type (**Text / Image / Video**)
3. Select a **Category** — placeholders will auto-update with best practices
4. Fill in the details
5. Click **"Generate JSON Prompt"**
6. Copy the prompt using the provider-specific buttons
